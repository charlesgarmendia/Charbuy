# 🌍 CharBuy Global - Ecosistema de Importación Inteligente
‎
‎**CharBuy Global** es una plataforma web progresiva diseñada para optimizar la experiencia de compra internacional. Centraliza herramientas críticas para importadores, compradores minoristas y entusiastas del comercio electrónico en una interfaz optimizada para dispositivos móviles y PC.
‎
‎## 🚀 Características Principales
‎
‎* **Buscador Dual Inteligente:** Búsqueda simultánea en los marketplaces más grandes del mundo (Amazon y Alibaba).
‎* **Conversor de Divisas Real-Time:** Sincronización automática con tipos de cambio globales sin dependencia de APIs restrictivas.
‎* **Calculadora de Importación (Land Cost):** Estimación inmediata de costos de envío y aranceles.
‎* **Rastreador Unificado:** Acceso directo al seguimiento de paquetes internacionales.
‎* **Diseño PWA:** Interfaz híbrida responsiva de alta velocidad bajo el ecosistema de Vercel.
‎
‎## 🛠️ Tecnologías Utilizadas
‎
‎* **Frontend:** HTML5, CSS3 (Modern Flexbox & Grid), JavaScript (ES6+).
‎* **APIs:** Integración con `currency-api` para datos financieros actualizados.
‎* **Deployment:** Vercel (Optimización de bordes y certificados SSL automáticos).
